# Phase 3 — 成長版本 (V1.5)

> **版本**：V1.5  
> **時程**：Month 8–10  
> **目標**：啟動病毒傳播引擎（分享卡片）+ 變現強化（虛擬道具/送溫暖）+ 聚合引擎 V2  
> **用戶規模目標**：澳門與香港合計 DAU 10,000 / MAU 50,000  
> **前置條件**：Phase 2 所有 Go/No-Go 項目通過，V1.0 穩定運行 ≥ 4 週  
> **測試地區**：澳門、香港（見 [test-regions-macao-hongkong.md](test-regions-macao-hongkong.md)）

---

## 一、階段目標

1. **病毒傳播引擎**：心情天氣分享卡片上線，驅動社群傳播
2. **變現強化**：虛擬道具 + 送溫暖機制上線，新增營收管道
3. **聚合引擎 V2**：時間衰減加權 + 動態排程，數據品質提升
4. **邀請機制**上線，擴大用戶基數
5. A/B Testing 框架就緒，建立數據驅動迭代能力
