# Phase 5 — 平台化 (V3.0)

> **版本**：V3.0  
> **時程**：Month 15+  
> **目標**：B2B 數據 API + 開發者生態 + 國際市場 + AI 預測 + SOC 2 合規準備  
> **用戶規模目標**：MAU 1,000,000（多國市場）  
> **前置條件**：Phase 4 所有 Go/No-Go 項目通過，MAU > 200,000

---

## 一、階段目標

1. **B2B 數據 API** 商業化，開拓企業/學術/政府客戶
2. **開發者生態**：開放 API 讓第三方整合 MoodMap 數據
3. **國際市場上線**：東京、台灣、東南亞等（澳門與香港已在 Phase 0–2 作為首發測試地區）
4. **AI 情緒趨勢預測**：預測區域情緒走向，為用戶和 B2B 客戶提供洞察
5. **SOC 2 Type II 合規準備**：為企業客戶建立信任

## 三、驗收標準

### 3.1 功能驗收

| # | 驗收項目 | 通過條件 | 驗證方式 |
|---|---------|---------|---------|
| A01 | B2B API | 所有端點回應正確、文件完整、Rate Limiting 正常 | API 測試套件 |
| A02 | API 數據安全 | API 回應中 100% 不含個人可識別資訊 | 自動化掃描 + 人工審查 |
| A03 | 開發者文件 | 文件涵蓋認證/端點/範例/錯誤碼，外部開發者可獨立使用 | 外部開發者試用回饋 |
| A04 | 東京上線 | 日文 App 完整上線 + 在地心理資源正確 + 環境情緒層就緒 | 手動測試 + 日文審查 |
| A05 | 台灣或東南亞上線 | 繁中/英文或當地語言 + 在地心情標籤 + 在地心理資源 | 手動測試 + 語言審查 |
| A06 | AI 預測模型 | 6h 預測準確率 > 70%，24h > 60%（情緒趨勢方向） | 模型回測 |
| A07 | 預測視覺化 | 地圖上趨勢箭頭/預測圖示正確顯示且直覺易懂 | 用戶測試 |
