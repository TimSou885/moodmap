# Phase 4 — 擴展版本 (V2.0)

> **版本**：V2.0  
> **時程**：Month 11–14  
> **目標**：多城市擴展 + 品牌合作 + 匿名對話 + 多語言 + Pro 方案  
> **用戶規模目標**：MAU 300,000（澳門＋香港深化，或加入台灣/廣東等擴展城市）  
> **前置條件**：Phase 3 所有 Go/No-Go 項目通過，澳門＋香港 DAU 穩定 > 8,000  
> **擴展方向**：香港各區深化、澳門全境、再擴台灣或華南（見 [test-regions-macao-hongkong.md](test-regions-macao-hongkong.md)）

---

## 一、階段目標

1. **多城市擴展**：高雄、台中、新北上線，每城市達到臨界密度
2. **品牌合作**：首批品牌打卡站 + 贊助罐頭回應上線
3. **匿名對話**（心情串聯）：讓用戶間產生更深連結
4. **多語言**：日/英語支援 + 多語言安全審核
5. **Pro 方案**上線，完善訂閱分層
6. 完成外部專家年度安全審計

---

## 二、交付項目

### 2.1 功能交付

| ID | 功能 | 描述 |
|----|------|------|
| F-010 | 城市心情報告 | 公開的城市情緒天氣報告，每日/每週自動產出 |
| F-012 | 品牌情緒打卡站 | 地圖上品牌 POI + 品牌專屬氣泡設計 |
| F-011 | 心情串聯（匿名對話） | 回覆心情文後可開啟匿名一對一對話（≤24h 時限） |
| — | 品牌贊助罐頭回應 | 品牌冠名回應（如「請你一杯咖啡 ☕」）+ 品牌贊助虛擬禮物 |
| — | 多語言支援 | 日文 / 英文 UI + 心情標籤 + 罐頭回應在地化 |
| — | Pro 方案 | NT$190/月：365 天歷程 + 城市報告 + 進階分析 + API |
| — | 節日特別版卡片 | 跨年 / 聖誕 / 農曆新年 / 情人節特別版分享卡片 |
| — | 多城市部署 | 高雄/台中/新北環境情緒層 + 預埋 + 在地 KOL 合作 |

### 2.2 安全防護

| 項目 | 內容 |
|------|------|
| 匿名對話安全掃描 | 對話內容即時四層審核 + 一鍵檢舉 + 封鎖對話方 |
| 對話專屬安全規則 | 連續負面對話偵測 → 插入關懷資源；對話中偵測高風險 → 紅色警戒 |
| 多語言安全審核 | 日/英語關鍵字庫 + NLP 模型多語言擴充 |
| 品牌內容審核 | 品牌相關內容不得出現在關懷資源/安全介面中 |
| 安全審計 #4 | 季度審查 + 外部專家年度審計 |
| 安全審計年報 | 彙整全年安全數據、事件、改善，向管理層報告 |

### 2.3 技術架構

| 項目 | 內容 |
|------|------|
| 多區域部署 | 為國際化準備的基礎設施架構 |
| i18n 框架 | React Native i18n + 後端多語言支援 |
| 對話服務 | Conversation Service + WebSocket 雙向即時通訊 |
| CDN | 靜態資源 + 分享卡片 CDN 加速 |
